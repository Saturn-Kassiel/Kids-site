# Интеграция модуля тестирования в «Гоша»

## Файлы модуля
- `testing.js` — логика (движок, задания, дашборд, результаты)
- `testing.css` — стили

## Шаги интеграции

### 1. Добавить файлы в корень проекта
Положить `testing.js` и `testing.css` рядом с `app.js` и `style.css`.

### 2. Подключить CSS в `<head>` (index.html)
```html
<link rel="stylesheet" href="testing.css">
```
После `<link rel="stylesheet" href="style.css">`.

### 3. Подключить JS перед `</body>` (index.html)
```html
<script src="testing.js"></script>
```
После `<script src="app.js"></script>`.

### 4. Добавить карточку в меню (index.html)
В `.menu-grid`, перед кнопкой «Достижения»:
```html
<button class="menu-card mc-testing" onclick="Testing.init()">
    <span class="mc-icon"><svg viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="1.8" stroke-linecap="round" stroke-linejoin="round"><path d="M9 11l3 3L22 4"/><path d="M21 12v7a2 2 0 0 1-2 2H5a2 2 0 0 1-2-2V5a2 2 0 0 1 2-2h11"/></svg></span>
    <span class="mc-label">Тестирование</span>
</button>
```

### 5. Добавить секцию (index.html)
Рядом с другими `<section>` (например, после секции `riddles`):
```html
<section id="testing" class="hidden">
    <div id="testing-content"></div>
</section>
```

### 6. (Опционально) Добавить в Service Worker
В `sw.js`, массив `SHELL_URLS`:
```js
'/Kids-site/testing.js',
'/Kids-site/testing.css',
```

## Что входит в v1

### Когнитивный блок
- Счёт предметов (3-4: до 5, 4-5: до 10, 5-6: до 15, 6-7: арифметика)
- Цвета (3-4: 4 цвета, до 5-6: 8 цветов)
- Формы (от 3 базовых до 7+ включая ромб и звезду)
- 4-й лишний (от очевидных категорий к абстрактным признакам)
- Последовательности (от 3 карточек до 6)

### Речевой блок
- Словарный запас (бытовые предметы → обобщения → антонимы → определения)
- Грамматика (с 5 лет — множественное число, согласование, предлоги)

### Моторный блок
- Монетки в копилку (drag & drop, скорость + точность)
- Копирование узора точек (тап по сетке)

### Социально-эмоциональный блок
- Распознавание эмоций (от 3 до 6 эмоций по возрасту)
- Социальные ситуации (по мотивам Банкова)
- Самообслуживание (чеклист)

### Инфраструктура
- Дашборд с паутинной диаграммой (canvas)
- Прогресс-бар во время теста
- Анимации переходов + confetti на финише
- localStorage: профиль + история результатов
- Полный тест / мини-тест по блокам
- Экран результатов с рекомендациями и red flags
